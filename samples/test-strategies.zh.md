# 测试策略集

一组专门用于测试 TradeTrace 全流程（Qwen 解析 → 证据收集 → 本地回测 → 风险评分 → 审批 → 执行）的自然语言策略。

每条都标注了**预期风险档位**，方便你针对性地测透不同分支：

- 🟢 **Go（低风险）** —— 可能直接放行执行
- 🟡 **Review（中风险）** —— 进入 `awaiting_approval`，Web 详情页 / Telegram 消息出现 Approve / Reject 按钮
- 🔴 **Block（高风险）** —— 风险引擎命中关键规则直接拦截

策略风格偏**口语化自然语言**（而非 `When X, go long. Use Y.` 那种模板腔），用来验证 Qwen 解析器对真实用户表达的鲁棒性。

> 另有 [`strategies.md`](strategies.md) —— 那是绑定两个 JSON fixture、用于**可复现性**的样本输入；本文件则是用于**测试覆盖**的策略清单，两者职责不同。

## 🟢 预期低风险（Go）

### 1. SOL 均值回归

```text
我观察到 SOL 在 15 分钟级别经常偏离 vwap 后又回归，所以想做这种回归：当价格跌到 vwap 下方 2 倍标准差以外时就开多，等它回到 vwap 就平。每次只用 5% 仓位，止损卡在 1%，同一时间只持一单，不贪心。
```

### 2. BTC 顺势加仓

```text
我想跟着大趋势做 BTC：只要 50 周期均线在 200 周期上方、RSI 没过热（40 到 60 之间）、并且 MACD 柱状图刚翻正，我就轻仓做多，止损 2%，止盈 5%，单笔仓位不超过 8%，一天最多做 4 单。
```

## 🟡 预期中风险（Review，触发审批按钮）

### 3. ETH 区间内低买高卖（网格）

```text
我觉得 ETH 最近会在 3000 到 3500 这个区间来回震荡，想在区间里做网格：每隔 2% 挂一档买单往下铺、每隔 2% 挂一档卖单往上铺，每档用 3 倍的小仓位，整个网格总敞口控制在 20% 以内。
```

> 网格策略 `unknowns` 较多，解析模糊 → 风险引擎常判 Review。

### 4. BTC 超跌反弹（带杠杆）

```text
当 BTC 在 1 小时图上 RSI 跌破 25，我判断是超跌了，想用 2 倍杠杆接多，等 RSI 回到 55 以上就走，目标 6% 利润，单次仓位 12% 左右。
```

> 2 倍杠杆 + 较大仓位 → 多半 Review。

### 5. 山寨币动量轮动

```text
我想做 24 小时内涨幅最强的那个山寨币，从 SOL、AVAX、LINK、DOT 里选最强的买进，持有 4 小时后换下一个最强的，每只最多放 10% 仓位，止损 3%。
```

## 🔴 预期高风险（Block，风险引擎拦截）

### 6. 翻倍加仓（马丁格尔 + 高杠杆）

```text
我每次亏了就把仓位翻一倍，直到赚回来为止，BTC 上用 10 倍杠杆，不设止损，全天候不停做。
```

### 7. 暴涨追击 all-in

```text
只要 ETH 一小时内涨了 5%，我就直接全仓杀进去，用 20 倍杠杆把收益放最大，不设止损，拿着等翻倍。
```

### 8. 越跌越买（摊低成本）

```text
SOL 只要是跌的我就一直加仓摊低成本，每跌 5% 就加 50% 的仓位，杠杆能开多高开多高。
```

## ⚙️ 专项 / 边界用例

### 9. 空头趋势

```text
我想在 BTC 走弱的时候做空：1 小时图上 EMA12 下穿 EMA26、同时 ADX 大于 25 说明趋势够强，我就进空单，止损 2%、止盈 4%、仓位 10%。
```

### 10. 极简模糊（测解析鲁棒性）

```text
就是低买高卖 ETH，你看着办。
```

### 11. 长线持仓

```text
我打算拿 BTC 做长线：只要周线收在 50 周均线上方，我就进场拿着至少 30 天，除非跌破 15% 止损。这笔可以放 25% 仓位。
```

### 12. 多空双向

```text
ETH 我多空都做，4 小时图上 EMA20 上穿 EMA50 就做多、下穿就做空，每笔止损 2%、每笔只冒 1.5% 的风险。
```

## 怎么跑

把任一条原文作为策略输入：

- **Web UI**：打开 `/runs/new`，粘贴到文本框，点 Create run。
- **Telegram**：`/run <粘贴上面的策略原文>`。
- **curl**：

```bash
curl -X POST http://localhost:3000/api/runs/stream \
  -H 'content-type: application/json' \
  -d '{"strategy":"我观察到 SOL 在 15 分钟级别经常偏离 vwap 后又回归……（粘贴原文）"}' \
  -m 120
```

> 需要 `QWEN_API_KEY`；回测走 Bitget 公共 klines，**不需要** Bitget key。
