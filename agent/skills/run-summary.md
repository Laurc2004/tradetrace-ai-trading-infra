# Run Summary Skill

A post-run report should be short, audit-friendly, and evidence-based.

Required sections:
1. Executive summary.
2. Strategy intent.
3. Backtest evidence.
4. Risk findings.
5. Approval decision.
6. Execution or replay result.
7. Recommended next actions.

Do not invent metrics. If a metric is unavailable, state that it is unavailable.
