# TradeTrace — 3 分钟演示视频脚本（中文版）

> 形式：**屏幕录制 + 中英双语字幕 + 中文 AI 配音**（你不亲自配音，用 TTS 生成）
> 硬性上限：**3:00**
> 配音：中文（用 TTS 生成）；字幕：中英双语（中文在上，英文在下）

本文是逐镜拍摄脚本。每行 = 一个镜头。录"画面"列的屏幕操作，把"中文配音"列喂给中文 TTS，"双语字幕"列烧到视频上。

各段时间预算（总和 ≤ 3:00）：

| 段 | 时长 | 累计 |
|---|---|---|
| ① 痛点 + 定位 | 0:25 | 0:25 |
| ② 成功路径（真实回测） | 1:15 | 1:40 |
| ③ 拦截路径（治理卖点） | 0:45 | 2:25 |
| ④ 低摩擦集成 + 收尾 | 0:35 | 3:00 |

---

## 录制前准备（务必先做，否则视频会很卡）

1. `npm run dev`，打开 `http://localhost:3000`。
2. **提前跑好两个 run**，录视频时不要现场等 Qwen（每次 50–90 秒）：
   - 提交成功策略（做空趋势 EMA）→ 跑完 → 点 **Approve**。
   - 提交危险策略（马丁格尔+高杠杆）→ 自动 **Block**。
   - 现在两个完成的 run 都在 dashboard 和详情页里，镜头切过去零等待。
3. 浏览器：全屏、缩放 110%、隐藏书签栏。桌面清干净。
4. 准备好标签页：`/runs/new`（空）、两个 run 详情页。
5. `data/runs.json` 只留这两个 run（或清空，靠 samples 演示）。

## 中文配音全文（直接喂中文 TTS，约 230 字 / 1:50）

> AI 交易智能体能力很强，但作为黑盒，它是危险的——你根本看不清它为什么做出某个决策。工具调用、回测、风控、审批，散落在各处，失败了也没法复盘。
>
> TradeTrace 补的就是这一层缺失的基础设施：让每一次运行都可观测、可治理、可回放。
>
> 从一条自然语言策略开始，Qwen 把它解析成结构化意图。飞行记录器捕获每一步——解析、证据包、回测、风险、审批、报告——汇成一条可回放的时间线。
>
> 关键是：回测是真的。它在 Bitget 官方公共 K线数据上做确定性模拟，不造假、不需要任何 key。
>
> 一个确定性的、基于规则的风险账本，用评审不用读代码也能懂的语言，讲清楚"为什么"。最后，一份运行后报告把一切串起来，引用的是真实的运行证据。
>
> 现在是最坏情况：一条危险策略——马丁格尔、高杠杆、没有止损。风险引擎打出 100 分满分，并在执行前就把它拦下——是确定性的规则，不是 LLM 猜的。这次运行根本到不了执行。完整的拦截路径被记录下来、可回放。
>
> 上手很低摩擦：一个依赖、一个 key——Qwen。回测用的是 Bitget 公共数据，不需要 Bitget key。通过 Web、API、或 Telegram 集成，三者共享同一份 run 存储。
>
> TradeTrace——为自主交易智能体而生的、可回放、可审计的治理层。GitHub 链接见简介。

按中文 TTS 正常语速（约 4 字/秒），230 字 ≈ 1:00–1:10，留足 ~1:50 给画面停顿、音乐和呼吸。✅

## 录制与制作清单

### 采集
- [ ] **OBS Studio**（免费）或 macOS Cmd+Shift+5 录屏。1920×1080，30fps。
- [ ] **按镜头分段录**——重做单个镜头比重做整条容易。
- [ ] 提前跑好两个 run（见"录制前准备"），镜头里不等 Qwen。
- [ ] 滚动慢、稳；每个关键数字停 1–2 秒，让字幕/配音对得上。

### 配音（AI / 中文 TTS）
- [ ] 把上面"中文配音全文"喂给中文 TTS（推荐：**Edge TTS 免费** / 微软晓晓 / ElevenLabs 多语种 / 豆包语音）。
- [ ] 选沉稳、中性的声音。生成一整段音频，再按段对齐。
- [ ] 可选：再用英文 TTS 生成一版英文配音，做英文受众的版本。

### 字幕（中英双语）
- [ ] 把"双语字幕"列按镜头时间码烧到画面上——**中文在上，英文在下**。
- [ ] 每条字幕尽量短，中文 ≤ 15 字，英文 ≤ 8 词。
- [ ] 也可用 Whisper 从中文配音自动生成时间轴，再人工校对+翻译。

### 剪辑
- [ ] DaVinci Resolve / 剪映 / iMovie。狠剪空档——3 分钟很快。
- [ ] 配音下垫轻量背景音乐，音量压低。
- [ ] 结尾卡：项目名 + GitHub 链接 + "Bitget AI Hackathon · Track 2 · Trading Infra"。

### 三个"别做"
- ❌ 别在镜头里现场等 Qwen（用预跑好的 run）。
- ❌ 别说"真实资金交易"（诚实说 replay / 模拟）。
- ❌ 别跳过真实回测数字（+8.3%、Sharpe 2.95、100/100 Block）——那是你的可验证证据亮点。
